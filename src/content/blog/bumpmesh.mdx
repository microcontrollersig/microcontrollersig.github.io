BumpMesh.com is a free, browser-based tool developed by CNC Kitchen.

It allows you to add custom textures to 3d models.

The motivation for this was figuring out the based way to create those cute knitted animals like
you see by MakerWorld creator <HyperLink href="https://makerworld.com/en/@TreeDprints">**TreeDprints**</HyperLink>
