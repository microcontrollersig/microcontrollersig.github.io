Here are steps to follow when can't upload ESPHome to your esp32.

## Prerequisites

### Is python installed?

```sh
python --version
```

### List python packages/libraries installed

```sh
python -m pip list
```

note if esphome and esptool are installed and what their version numbers are.

### Install/update esphome and esptool cli

```sh
python -m pip install esptool esphome
```

you can confirm install by re-running:

```sh
python -m pip list
```

1. the official package manager for downloading libraries is: **pip**.
1. the official registry/place where it downloads libraries from is: **pypi**.
1. link to esptool pypi page <HyperLink href="https://pypi.org/project/esptool">here</HyperLink> 
1. link to esphome pypi page <HyperLink href="https://pypi.org/project/esphome">here</HyperLink> 




## EXECUTIVE SUMMARY OF PROCEDURE

### Erase Flash 
   ```sh 
   esptool erase_flash --device COM7
   ```
### Upload firmware via USB 
  Need to upload esphome yaml via USB after erase. 
  
  Once esphome is on device, over-the-air upload will work.

  ```sh
   esphome compile something.yaml
   ```

   Followed by:

   ```sh
   esphome upload something.yaml --device COM7
   ```

Now uploading from Home Assistant ESPHome Addon should work.
